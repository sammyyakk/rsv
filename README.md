# RSV Drug Discovery — AI-Assisted Molecular Docking

This repository contains the planning and execution documents for an AI-assisted
drug-discovery project targeting **Respiratory Syncytial Virus (RSV)** using free
and open-source molecular docking tools (AutoDock Vina and compatible platforms).
